# WanX 图生视频 - Mac 安装指南

欢迎使用 WanX 图生视频应用！

## 📥 安装步骤

### 1. 下载应用
下载 `WanX.dmg` 文件到您的 Mac

### 2. 安装应用
1. 双击 `WanX.dmg` 文件
2. 将 `WanX.app` 拖拽到 `Applications` 文件夹
3. 等待拷贝完成

### 3. 首次运行

**方法一：从 Launchpad 启动**
- 打开 Launchpad
- 找到 WanX 图标
- 点击启动

**方法二：从 Applications 启动**
- 打开 Finder
- 前往 Applications 文件夹
- 双击 WanX

## ⚠️ 安全提示

首次运行时，macOS 可能显示安全警告：

```
"WanX.app" 无法打开，因为它来自未验证的开发者
```

### 解决方法

**推荐方式：**
1. 右键点击 `WanX.app`
2. 选择"打开"
3. 在弹出的对话框中点击"打开"按钮
4. 以后就可以正常双击启动了

**或者使用命令行：**
```bash
# 打开终端，执行：
xattr -cr /Applications/WanX.app
```

**或者在系统设置中：**
1. 打开"系统偏好设置"
2. 进入"安全性与隐私"
3. 在"通用"标签页中
4. 点击"仍要打开"按钮

## 🔑 配置 API 密钥

首次使用需要配置 DashScope API 密钥：

1. 启动 WanX 应用
2. 点击右上角的"设置"按钮⚙️
3. 在"API 配置"中填入您的密钥
4. 点击"保存"

**获取 API 密钥：**
- 访问 [DashScope 控制台](https://dashscope.console.aliyun.com/)
- 登录后获取 API 密钥

## 🚀 开始使用

### 创建工程
1. 点击"新建工程"
2. 输入工程名称
3. 选择保存位置

### 文生图功能
1. 点击"🎨 文生图"标签
2. 输入描述文本
3. 选择模型和尺寸
4. 点击"生成图片"

### 图生视频功能
1. 点击"📹 图生视频"标签
2. 上传或选择图片
3. 输入视频描述
4. 选择模型和参数
5. 点击"生成视频"

## 📂 数据位置

应用数据保存在：
```
~/Library/Application Support/WanX/
```

包含：
- 工程文件
- 上传的图片
- 生成的视频
- 配置文件

## ❓ 常见问题

### Q1: 应用无法打开？
**A:** 请按照"安全提示"部分的说明操作

### Q2: 提示 API 密钥错误？
**A:** 
- 检查密钥是否正确
- 确认密钥有效期
- 检查网络连接

### Q3: 生成失败？
**A:**
- 检查网络连接
- 查看错误提示
- 确认 API 额度充足

### Q4: 如何卸载？
**A:**
1. 删除 `/Applications/WanX.app`
2. 删除 `~/Library/Application Support/WanX`（可选）

### Q5: 应用占用空间太大？
**A:** 应用约 400MB，包含：
- Python 运行时
- PyQt5 框架
- 所有依赖库
