# 万相2.6 文生图模型快速指南

## 🌟 模型特性

万相2.6文生图（wan2.6-t2i）是最新发布的文字生成图片模型，提供高质量的图像生成能力。

### 核心特性

- **灵活分辨率**：支持自定义宽高，总像素在 [768×768, 1440×1440] 之间
- **宽高比范围**：支持 [1:4, 4:1] 的宽高比
- **输出格式**：PNG 格式，保证图像质量
- **调用方式**：与万相2.5相同，无需额外适配

## 📋 参数说明

### 模型标识
```
model: "wan2.6-t2i"
```

### 图像分辨率

**总像素约束**
- 最小：768 × 768 = 589,824 像素
- 最大：1440 × 1440 = 2,073,600 像素

**宽高比约束**
- 最窄：1:4（例如 720×2880）
- 最宽：4:1（例如 2880×720）
- 推荐：1:1 到 16:9 之间

### 常用分辨率预设

| 比例 | 分辨率 | 用途 |
|------|--------|------|
| 1:1 | 1280×1280 | 社交媒体、头像 |

## ⚠️ 注意事项

### 1. 分辨率限制

**有效的分辨率：**
- ✅ 1280×1280（总像素：1,638,400）
- ✅ 1440×810（总像素：1,166,400）
- ✅ 1024×1024（总像素：1,048,576）

**无效的分辨率：**
- ❌ 512×512（总像素太小：262,144）
- ❌ 2048×2048（总像素太大：4,194,304）
- ❌ 1440×360（宽高比超出范围：4:1）

### 2. 宽高比计算

确保宽高比在 [1:4, 4:1] 范围内：
```
宽高比 = 宽度 / 高度

有效范围：
0.25 ≤ 宽高比 ≤ 4.0

示例：
- 720×1440 = 0.5 ✅
- 1440×720 = 2.0 ✅
- 720×2880 = 0.25 ✅
- 2880×720 = 4.0 ✅
- 720×3000 = 0.24 ❌（超出范围）
```

### 3. 输出格式

- 所有图片以 PNG 格式输出
- 保证无损质量
- 支持透明通道（如果生成内容包含）

## 📞 技术支持

### 常见问题

**Q: 为什么我的分辨率设置无效？**
A: 检查总像素是否在 [768², 1440²] 范围内，宽高比是否在 [1:4, 4:1] 范围内。

**Q: 如何提高生成质量？**
A: 
1. 使用详细的提示词
2. 启用提示词扩展
3. 添加质量相关的关键词（如"高清"、"细节丰富"）
4. 使用反向提示词排除不想要的元素

**Q: 生成的图片不符合预期怎么办？**
A:
1. 调整提示词，更具体地描述想要的效果
2. 尝试不同的随机种子
3. 使用反向提示词排除不想要的元素
