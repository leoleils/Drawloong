# 🎉 v1.3.0 更新 - 拖拽与文件浏览

## 🌟 主要新功能

### 1️⃣ 图片查看器 🖼️
**双击图片，立即预览！**
- ✨ 高质量缩放（放大/缩小）
- 🎯 适应窗口/实际大小
- ⌨️ 快捷键支持（Ctrl +/-/0/1）
- 📊 实时显示缩放比例

### 2️⃣ 视频播放 🎬
**双击视频，即刻观看！**
- 🎥 使用系统默认播放器
- 🚀 不占用客户端资源
- 📺 支持 MP4、AVI、MOV

### 3️⃣ 拖拽导入 🔄
**拖拖拽拽，轻松导入！**

**支持三种拖拽方式：**

#### 方式一：外部 → 资源管理器
```
系统文件夹的文件
    ↓ 拖拽
资源管理器
    ↓ 自动识别
图片 → inputs/
视频 → outputs/
```

#### 方式二：外部 → 上传区域
```
系统文件夹的图片
    ↓ 拖拽
上传区域
    ↓
自动加载预览
```

#### 方式三：资源管理器 → 上传区域
```
inputs/ 文件夹的图片
    ↓ 双击查看
    ↓ 关闭查看器
自动加载到上传区域
```

## 🎯 快速体验

### 场景 1：查看图片
1. 双击资源管理器中的图片 🖼️
2. 图片查看器打开
3. 使用工具栏或快捷键缩放
4. 按 ESC 关闭

### 场景 2：播放视频
1. 双击资源管理器中的视频 🎬
2. 系统播放器自动打开
3. 欣赏生成的视频

### 场景 3：导入素材
1. 从桌面拖动图片
2. 放到资源管理器
3. 自动复制到 inputs/
4. 立即可用

### 场景 4：快速选图
1. 双击 inputs 中的图片
2. 预览后关闭查看器
3. 图片自动加载到上传区
4. 直接配置参数生成

## ⌨️ 快捷键

### 图片查看器
| 功能 | Windows/Linux | macOS |
|------|--------------|-------|
| 放大 | Ctrl + + | Cmd + + |
| 缩小 | Ctrl + - | Cmd + - |
| 适应窗口 | Ctrl + 0 | Cmd + 0 |
| 实际大小 | Ctrl + 1 | Cmd + 1 |
| 关闭 | ESC | ESC |

## 💡 使用技巧

### 提高效率
✅ 双击图片快速预览，无需外部软件  
✅ 从桌面直接拖拽导入，无需复制粘贴  
✅ inputs 图片双击后自动加载，减少点击  
✅ 视频生成完双击查看，即时验证效果  

### 工作流优化
```
1. 准备素材：拖拽图片到资源管理器
2. 选择图片：双击预览 → 关闭（自动加载）
3. 配置参数：设置提示词等
4. 生成视频：点击生成
5. 查看结果：双击视频播放
6. 继续处理：返回步骤2
```

## ⚠️ 注意事项

1. **图片查看器**
   - 建议图片不超过 10MB
   - 过大图片可能加载较慢

2. **视频播放**
   - 需要系统安装播放器
   - macOS 默认有 QuickTime
   - Windows/Linux 建议安装 VLC

3. **拖拽导入**
   - 必须先打开工程
   - 只支持指定格式文件
