# 万相2.5图像编辑快速指南

## 🌟 新功能亮点

图像编辑功能现已支持 **万相2.5模型 (wan2.5-i2i-preview)**,提供更优质的编辑效果!

## 🚀 快速开始

### 单图编辑示例

**场景**: 换装编辑

1. 打开应用,选择"图像编辑"标签页
2. 模型选择: **🌟 wan2.5-i2i-preview（推荐）**
3. 编辑模式: 🖼️ 单图编辑
4. 添加图片: 选择1张人物照片
5. 编辑描述:
   ```
   将花卉连衣裙换成一件复古风格的蕾丝长裙，
   领口和袖口有精致的刺绣细节。
   ```
6. 生成数量: 2张
7. 点击"开始编辑"
8. 等待处理 (约30-60秒)
9. 查看结果

### 多图融合示例

**场景**: 物体迁移

1. 打开应用,选择"图像编辑"标签页
2. 模型选择: **🌟 wan2.5-i2i-preview（推荐）**
3. 编辑模式: 🎭 多图融合
4. 添加图片:
   - 图1: 物体照片 (如闹钟)
   - 图2: 场景照片 (如餐桌)
5. 编辑描述:
   ```
   将图1中的闹钟放置到图2的餐桌的花瓶旁边位置
   ```
6. 生成数量: 2张
7. 点击"开始编辑"
8. 等待处理 (约30-60秒)
9. 查看结果

## ❓ 常见问题

### Q: 为什么万相2.5不支持反向提示词?
**A**: 万相2.5模型的API设计中不包含反向提示词参数,但效果已经很优秀,无需依赖反向提示词。

### Q: 处理时间多长?
**A**: 
- 单图编辑: 约20-40秒
- 多图融合: 约30-60秒
- 具体时间取决于服务器负载

### Q: 如果超时怎么办?
**A**: 
1. 检查网络连接
2. 重新提交任务
3. 减少生成数量
4. 稍后重试

### Q: 生成的图片保存在哪里?
**A**: 保存在当前工程的 `inputs` 文件夹中,文件名格式为 `image_edit_YYYYMMDD_HHMMSS_N.png`

### 4. 合理配额

- 先用少量图片测试

## 🔧 故障排除

### 问题: 提交失败
**解决方案**:
1. 检查API密钥是否正确
2. 确认网络连接正常
3. 验证图片格式是否支持

### 问题: 一直处理中
**解决方案**:
1. 耐心等待 (最多2分钟)
2. 检查状态标签的提示
3. 如果超时会自动提示

### 问题: 结果不理想
**解决方案**:
1. 调整提示词描述
2. 更换输入图片
3. 增加生成数量
4. 尝试多次生成
