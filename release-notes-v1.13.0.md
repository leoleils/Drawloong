# 烛龙绘影 v1.13.0 发布说明

发布日期：2025-12-15

## 🎉 新功能

### 首尾帧页面优化
- **图片预览区域增大**：首帧和尾帧预览区域高度从150px增加到220px，图片显示更清晰，不再显得扁平

### 视频播放功能增强
- **智能视频播放**：在首尾帧页面点击视频集中的视频时，会在该页面的视频浏览器中播放，而不是跳转到其他页面
- 根据当前激活的标签页自动选择对应的视频播放器

### 资源管理器全面优化

#### 界面优化
- **隐藏分割器手柄**：去除资源管理器旁边的蓝色分割条，界面更简洁美观
- **刷新按钮样式优化**：透明背景设计，悬停时显示浅灰色背景

#### 文件过滤功能
- **图集专注显示图片**：图集文件夹只显示 .png、.jpg、.jpeg 格式的图片文件
- **视频集专注显示视频**：视频集文件夹只显示 .mp4 格式的视频文件
- 自动过滤不相关的文件类型，让资源管理更清晰

#### 视频预览图功能 🆕
- **视频缩略图显示**：视频集中的视频文件现在会显示第一帧作为预览图
- 使用 opencv-python 库提取视频第一帧
- 48x48 像素的高质量缩略图
- 如果提取失败，会优雅降级显示文件名

## 🐛 Bug 修复

- **修复 Task 对象属性错误**：将错误的 `output_file` 属性改为正确的 `output_path`
- 修复任务完成后自动加载视频的功能

## 🔧 技术改进

- 新增依赖：opencv-python (用于视频预览图提取)
- 优化 QSplitter 样式，手柄宽度设置为0
- 改进文件过滤逻辑，根据文件夹类型智能过滤

## 📦 依赖更新

```bash
pip install opencv-python
```

## 🎯 使用建议

1. **视频预览图**：首次打开包含大量视频的工程时，预览图提取可能需要几秒钟
2. **文件管理**：现在图集和视频集各司其职，建议将图片放入图集，视频放入视频集
3. **视频播放**：在首尾帧页面工作时，可以直接点击视频集中的视频预览
