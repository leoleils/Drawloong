# 文件类型控制更新

## 更新内容

实现了图集和视频集的严格文件类型控制：

### 图集（inputs 文件夹）
- **只能添加图片文件**：.png, .jpg, .jpeg, .bmp, .gif, .webp
- **只显示图片文件**：视频文件会被自动过滤，不在列表中显示
- 拖拽导入时，图片文件自动导入到图集

### 视频集（outputs 文件夹）
- **只能添加视频文件**：.mp4, .avi, .mov, .mkv, .flv, .wmv
- **只显示视频文件**：图片文件会被自动过滤，不在列表中显示
- 拖拽导入时，视频文件自动导入到视频集

## 技术实现

### 1. 显示过滤（load_folder 方法）
- 图集：只显示图片扩展名的文件
- 视频集：只显示视频扩展名的文件
- 其他类型文件会被自动跳过，不显示在列表中

### 2. 导入控制（import_file 方法）
- 图片文件自动导入到 inputs 文件夹（图集）
- 视频文件自动导入到 outputs 文件夹（视频集）
- 不支持的文件类型会显示错误提示

### 3. 用户体验
- 批量导入时显示导入结果统计
- 不支持的文件类型会显示清晰的错误信息
- 文件自动分类到正确的文件夹

## 支持的文件格式

**图片格式**：
- .png
- .jpg / .jpeg
- .bmp
- .gif
- .webp

**视频格式**：
- .mp4
- .avi
- .mov
- .mkv
- .flv
- .wmv

## 使用说明

1. 拖拽文件到资源管理器时，系统会自动识别文件类型
2. 图片文件会自动添加到"图集"
3. 视频文件会自动添加到"视频集"
4. 不支持的文件类型会显示错误提示
5. 每个文件夹只显示对应类型的文件
