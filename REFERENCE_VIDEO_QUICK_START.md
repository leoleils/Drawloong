# 参考生视频快速入门

## 简介

参考生视频功能允许你上传1-2个参考视频，通过文本描述生成新的视频内容。系统会识别参考视频中的主体（角色、物体等），并根据你的描述生成新场景。

## 快速开始

### 第一步：准备参考视频

确保你的参考视频符合以下要求：

- ✅ 格式：mp4 或 mov
- ✅ 时长：2-30秒
- ✅ 大小：每个视频不超过100MB
- ✅ 内容：主体清晰、背景简洁

**推荐视频特征：**
- 主体占据画面中心位置
- 光线充足，画面清晰
- 避免过多背景干扰
- 动作自然流畅

### 第二步：打开参考生视频功能

1. 启动烛龙绘影应用
2. 创建或打开一个工程
3. 点击顶部的"参考生视频"标签页

### 第三步：上传参考视频

#### 方式一：点击选择
1. 点击"选择视频"按钮
2. 从文件浏览器中选择视频文件
3. 确认上传

#### 方式二：拖拽上传
1. 直接将视频文件拖拽到预览区域
2. 系统自动识别并加载

**提示：**
- 参考视频1（character1）：必选
- 参考视频2（character2）：可选，用于多角色场景

### 第四步：编写提示词

在"视频描述"框中输入你的创意：

#### 单个参考视频示例：
```
character1在沙发上开心地看电影
```

```
character1在公园里慢跑，阳光洒在身上
```

```
character1站在海边，微风吹拂着头发
```

#### 多个参考视频示例：
```
character1和character2在咖啡厅里愉快地交谈
```

```
character1向character2递出一束鲜花
```

**重要提示：**
- 必须使用 `character1` 来指代第一个参考视频中的主体
- 如果有第二个参考视频，使用 `character2` 来指代
- 提示词要具体、清晰，避免模糊描述

### 第五步：配置生成参数

#### 分辨率选择
根据你的需求选择合适的分辨率：

**720P档位（推荐新手）：**
- 16:9 横屏 (1280*720) - 适合横屏视频
- 9:16 竖屏 (720*1280) - 适合手机竖屏
- 1:1 方形 (960*960) - 适合社交媒体

**1080P档位（高清）：**
- 16:9 横屏 (1920*1080) - 高清横屏
- 9:16 竖屏 (1080*1920) - 高清竖屏
- 1:1 方形 (1440*1440) - 高清方形

#### 视频时长
- **5秒**：快速预览，适合简单动作
- **10秒**：标准时长，适合完整叙事

#### 镜头类型
- **单镜头**：保持固定视角，画面稳定
- **多镜头**：自动切换角度，更有电影感

#### 音频选项
- ✅ 包含音频：生成带背景音的视频
- ❌ 不包含音频：生成静音视频

### 第六步：开始生成

1. 检查所有配置是否正确
2. 点击"开始生成"按钮
3. 等待视频生成（通常需要3-5分钟）

**生成过程：**
```
📤 正在上传参考视频...
📤 正在提交任务...
⏳ 任务已提交 (ID: xxx)
🔄 正在生成视频... (1/180)
📥 正在下载视频...
✅ 视频生成成功！
```

### 第七步：查看结果

生成完成后：
- 视频会自动在右侧预览区域播放
- 视频文件保存在工程的 `outputs` 文件夹中
- 可以在工程资源管理器中找到生成的视频

## 实用技巧

### 提示词编写技巧

#### ✅ 好的提示词
```
character1穿着红色连衣裙，在花园里优雅地旋转
```
- 具体描述动作
- 包含场景信息
- 有视觉细节

#### ❌ 不好的提示词
```
character1很开心
```
- 过于抽象
- 缺少动作描述
- 没有场景信息

### 反向提示词使用

在"反向提示词"框中输入不希望出现的内容：

```
低分辨率、模糊、变形、多余的手指、比例不良
```

这可以帮助提高生成质量，避免常见问题。

### 参考视频选择建议

#### 适合的参考视频：
- ✅ 人物特写或半身像
- ✅ 单一主体，背景简洁
- ✅ 光线均匀，画面清晰
- ✅ 动作自然流畅

#### 不适合的参考视频：
- ❌ 多人混乱场景
- ❌ 主体不清晰
- ❌ 光线过暗或过曝
- ❌ 快速运动模糊

### 多角色场景技巧

使用两个参考视频时：

1. **确保两个视频风格一致**
   - 相似的光线条件
   - 相似的拍摄角度
   - 相似的画面质量

2. **明确角色关系**
   ```
   character1（男性）向character2（女性）递出礼物
   ```

3. **描述互动动作**
   ```
   character1和character2面对面交谈，表情愉快
   ```

## 常见问题

### Q1: 为什么上传视频失败？
**A:** 检查以下几点：
- 视频格式是否为mp4或mov
- 文件大小是否超过100MB
- 视频时长是否在2-30秒之间
- 网络连接是否正常

### Q2: 生成的视频与预期不符？
**A:** 尝试以下方法：
- 使用更具体、详细的提示词
- 添加反向提示词排除不想要的内容
- 选择更清晰的参考视频
- 调整镜头类型（单镜头/多镜头）

### Q3: 生成时间太长？
**A:** 视频生成通常需要3-5分钟，如果超过10分钟：
- 检查网络连接
- 查看任务列表中的状态
- 如果失败，查看错误信息

### Q4: character1没有生效？
**A:** 确保：
- 提示词中正确使用了 `character1` 关键字
- 关键字拼写正确（全小写）
- 参考视频已成功上传

### Q5: 如何提高生成质量？
**A:** 最佳实践：
- 使用高质量的参考视频
- 编写详细、具体的提示词
- 选择合适的分辨率和时长
- 使用反向提示词
- 多次尝试，调整参数
