# 🎉 v1.2.0 更新说明

## 主要更新

本次更新为图生视频 QT 客户端带来了**工程管理功能**，让您能够像使用 VSCode 一样管理多个视频项目！

## 🌟 新功能一览

### 1️⃣ 工程管理系统
- ✨ 创建和管理多个独立工程
- 📂 每个工程自动组织输入/输出文件
- 🕐 最近使用工程快速访问
- 💾 独立的任务历史记录

### 2️⃣ 全新界面布局
- 🎨 类似 VSCode 的三栏式工作区
- 👋 友好的欢迎页面
- 🌲 文件树形资源管理器
- 🔧 可调整大小的面板

### 3️⃣ 自动文件组织
- 📥 上传的图片自动保存到 `inputs/` 文件夹
- 📤 生成的视频自动保存到 `outputs/` 文件夹
- 🔄 实时同步显示文件变化

## 🚀 快速开始

### 第一步：启动应用
```bash
cd qt_client
python main.py
```

### 第二步：创建工程
1. 在欢迎页面点击 **"新建工程"**
2. 输入工程名称（如："我的视频项目"）
3. 选择保存位置
4. 点击"创建"

### 第三步：开始创作
1. 拖拽图片到上传区域
2. 配置生成参数
3. 点击"生成视频"
4. 视频会自动保存到工程的 `outputs/` 文件夹

## 📁 工程结构

每个工程会自动创建以下结构：

```
我的工程/
├── inputs/          # 输入图片文件夹
├── outputs/         # 输出视频文件夹
├── project.json     # 工程配置
└── tasks.json      # 任务历史
```

## 💡 实用技巧

### 快捷键
- `Ctrl+Shift+N` - 新建工程
- `Ctrl+O` - 打开工程
- `Ctrl+,` - 打开设置

### 从资源管理器加载图片
双击左侧资源管理器中的图片，会自动加载到上传区域

### 批量处理
将多张图片放入 `inputs/` 文件夹，在资源管理器中依次点击处理

## ❓ 常见问题

**Q: 为什么必须创建工程才能使用？**  
A: 工程管理帮助您更好地组织文件，让素材和成果不会散落各处。

**Q: 可以在不同电脑上使用同一个工程吗？**  
A: 可以！只需复制整个工程文件夹即可。

**Q: 如何删除工程？**  
A: 目前需要手动删除工程文件夹，未来版本会添加删除功能。

**Q: 最近使用列表能显示多少个工程？**  
A: 默认显示最近使用的 10 个工程。

## 🔄 从旧版本升级

如果您之前使用过本客户端：

1. **API 密钥**: 已配置的密钥会自动保留
2. **旧任务**: 根目录的 `tasks.json` 保持不变
3. **新功能**: 现在需要创建工程才能生成新视频
4. **建议**: 为旧任务创建一个归档工程

## 🐛 已知问题

- macOS 启动时可能显示 "error messaging the mach port" 警告（可忽略）
- 暂不支持工程重命名（需手动编辑配置文件）
- 暂不支持在界面中删除工程（需手动删除文件夹）
