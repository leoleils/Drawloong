# 烛龙绘影 v1.17.1 发布说明

## 🚀 图像编辑功能增强

### 新增功能

#### 🆕 通义千问 Edit Plus 推荐模型
- **最新模型**: 新增 qwen-image-edit-plus 模型，标记为"推荐"
- **批量输出**: 支持 1-6 张图片批量生成
- **自定义分辨率**: 支持9种常用宽高比选择
- **智能优化**: 支持提示词智能改写功能
- **反向提示词**: 支持排除不需要的元素

### 技术改进

#### 🎯 智能UI适配
- **动态界面**: 根据选择的模型自动显示/隐藏相关功能选项
- **模型说明**: 为每个模型提供详细的功能描述
- **用户体验**: 避免用户设置不支持的参数组合

#### ⚡ API优化
- **参数支持**: 为 qwen-image-edit-plus 模型添加 size 参数支持
- **向后兼容**: 保持与现有模型的完全兼容性
- **错误处理**: 统一的错误处理机制

## 🎨 支持的分辨率

### 常用宽高比
- **1:1** - 1280×1280, 1024×1024
- **2:3** - 800×1200 (竖屏)
- **3:2** - 1200×800 (横屏)
- **3:4** - 960×1280 (竖屏)
- **4:3** - 1280×960 (横屏)
- **9:16** - 720×1280 (手机竖屏)
- **16:9** - 1280×720 (宽屏)
- **21:9** - 1344×576 (超宽屏)

## 🎯 使用建议

### 推荐使用场景
- **单图编辑**: 选择 qwen-image-edit-plus，设置合适的输出尺寸
- **多图融合**: 选择 qwen-image-edit-plus，可生成多张不同效果的图片
- **批量生成**: 设置 n=4-6，一次生成多个变体进行选择

### 最佳实践
1. **提示词优化**: 启用智能改写功能，提升生成效果
2. **尺寸选择**: 根据用途选择合适的宽高比
3. **反向提示词**: 使用反向提示词排除不需要的元素
4. **批量对比**: 生成多张图片进行效果对比
