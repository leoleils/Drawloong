# 万相2.6 模型快速指南

## 🌟 模型特性

万相2.6 是最新发布的图生视频模型，相比之前的版本有以下重要更新：

### 核心特性

- **更长时长支持**：支持 5秒、10秒、15秒 三种时长选择
- **高清分辨率**：支持 720P 和 1080P（不再支持480P）
- **智能多镜头**：新增镜头类型选择功能
- **30fps 流畅度**：保持高帧率输出

## 📋 参数说明

### 模型标识
```
model: "wan2.6-i2v"
```

### 分辨率 (resolution)
- ✅ `720P` (1280x720)
- ✅ `1080P` (1920x1080)
- ❌ ~~`480P`~~ (不再支持)

### 视频时长 (duration)
- ✅ `5` 秒
- ✅ `10` 秒
- ✅ `15` 秒（新增）

### 镜头类型 (shot_type)
**重要**：仅在启用提示词扩展时生效

- `multi` - 智能多镜头（默认推荐）
  - 自动分析场景，智能切换镜头
  - 适合复杂场景和动态内容
  
- `single` - 单镜头生成
  - 保持单一镜头视角
  - 适合简单场景和固定视角

## 🎯 使用建议

### 1. 选择合适的时长

- **5秒**：快速预览，适合简单动作
- **10秒**：标准时长，适合大多数场景
- **15秒**：完整叙事，适合复杂场景

### 2. 镜头类型选择

#### 推荐使用多镜头的场景：
- 人物动作丰富的场景
- 需要展示多个角度的内容
- 叙事性强的视频

#### 推荐使用单镜头的场景：
- 固定视角的场景
- 简单的动态效果
- 需要保持视角一致性

### 3. 分辨率选择

- **720P**：生成速度快，适合快速预览
- **1080P**：高清画质，适合最终输出

## 💡 最佳实践

### 提示词建议

1. **启用提示词扩展**（推荐）
   - 让AI优化你的描述
   - 自动添加镜头运动细节
   - 提升视频质量

2. **详细描述场景**
   ```
   好的提示词示例：
   "一只橘猫在阳光明媚的花园里追逐蝴蝶，镜头从远景推进到特写，展现猫咪好奇的表情"
   ```

3. **指定镜头运动**
   ```
   包含镜头运动的提示词：
   "镜头缓慢推进，展现城市夜景的霓虹灯光"
   "镜头环绕拍摄，360度展示产品细节"
   ```

### 参数组合建议

#### 快速预览模式
- 时长：5秒
- 分辨率：720P
- 镜头：单镜头
- 适合：快速测试想法

#### 标准制作模式
- 时长：10秒
- 分辨率：1080P
- 镜头：多镜头
- 适合：大多数创作需求

#### 专业制作模式
- 时长：15秒
- 分辨率：1080P
- 镜头：多镜头
- 提示词扩展：开启
- 适合：高质量输出

## ⚠️ 注意事项

1. **分辨率限制**
   - 2.6模型不支持480P
   - 如需480P，请使用2.2或2.5模型

2. **镜头类型生效条件**
   - 必须启用"提示词智能改写"
   - 关闭扩写时，镜头类型参数不生效

3. **生成时间**
   - 15秒视频生成时间较长
   - 建议先用5秒测试效果
